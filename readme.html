1 快速上手三步驟

1設定 API Key

本程式使用 Google Gemini API。首次使用請點擊左上角「常用設定」中的「API 金鑰」，貼上您的 Key 並儲存。
(金鑰僅暫存於您的瀏覽器，有效期限 2 小時，安全無虞)

👉 點此申請免費 API Key

2提供內容素材

在左側「提供內容」區塊，您可以：

  • 貼上文字：直接複製貼上教學講義。
  • 上傳圖片：拍下課本或講義，AI 會自動讀圖。
  • 貼上網址：支援一般網頁或 YouTube 影片(需字幕)。
  • AI 生成：輸入主題（如「光合作用」），讓 AI 幫您寫文章！

3設定並出題

確認下方的「題目生成設定」：

  • 填寫 試卷標題 (選填)
  • 選擇 學生程度 (這很重要！影響題目深淺)
  • 選擇 匯出格式 (如 PDF 考卷、Wordwall 等)
  • 設定 題目數量 與 類型

最後按下 開始出題，AI 就會開始工作囉！

2 進階功能詳解

✨ AI 生成學習內文

沒有現成的文章？沒問題！切換到 「✨ AI 生成內容」 分頁。

  • 輸入簡單的主題關鍵字 (如：台灣地理)。
  • 選擇 「學生程度」，AI 會自動調整文章難易度。
  • 勾選 「素養導向」，AI 會撰寫更具生活情境的文章。
  • 甚至可以自訂「寫作語氣」（如：活潑有趣、嚴謹學術）。

📄 PDF 考卷與 TXT 匯出

想要傳統紙筆測驗？QuestWiz 也能辦到！

  • 在匯出格式選擇 「PDF 考卷 (A4)」。
  • 填寫 「試卷標題」 (選填)，這會成為考卷的大標題與檔名。
  • 系統會自動生成包含卷頭（班級/姓名/座號/得分）的 A4 格式考卷供下載列印。
  • 若需要純文字以便在 Word 編輯，可選擇 「純文字檔 (.txt)」，包含題目與最後的解答列表。

🔗 內容分享功能

課堂即時閱讀神器！

生成或貼上文章後，點擊輸入框下方的 「分享內容」 按鈕。系統會產生一個專屬的 QR Code 與連結，學生掃描後即可在手機上閱讀該篇文章（純閱讀模式，無干擾）。

? 常見問題 (FAQ)

Q: 為什麼我匯入 PDF 檔案後，文字框卻是空白的？

這通常是因為該 PDF 檔案是由圖片或是掃描文件組成的（沒有文字層）。目前的技術只能讀取包含可選取文字的 PDF。建議您使用 OCR 工具將圖片轉為文字後再貼上。

注意：由此程式生成的 PDF 考卷也是圖片格式，因此無法再次被本程式讀取喔！

Q: API Key 會被偷走嗎？安全嗎？

絕對安全。您的 API Key 只會儲存在您當前使用的瀏覽器暫存區 (Session Storage) 中，並且我們設定了 2 小時自動過期機制。金鑰只會直接傳送給 Google 伺服器進行驗證，不會經過我們的後台，也不會被他人看見。

Q: 為什麼有時候生成的題目會變少？

這可能受限於 AI 的單次輸出長度限制，或者您提供的文章內容較短，AI 難以從中挖掘出足夠數量的題目。建議您可以分批生成（例如一次生 5 題），或是提供更豐富的內文素材。
