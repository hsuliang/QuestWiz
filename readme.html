快速上手指南

1

設定金鑰

點擊右上角「設定」，貼入您的 Google Gemini API Key。您可以一次貼入多組金鑰（以換行分隔），系統會自動輪替使用以確保最佳出題效率。

2

提供素材

貼上文字、上傳講義照片，或直接輸入主題讓 AI 為您撰寫教學文章。

3

調整參數與出題

選擇年級、出題風格與層次後點擊「開始出題」，AI 將在數秒內為您交卷。

4

預覽與分享

在預覽區進行最後校對與排序，接著下載檔案或上傳至題庫大廳分享成果。

常見問題

Q: 為什麼我有時候會遇到「配額不足」的錯誤？

A: Google API 有每日及每分鐘的使用量限制。建議您申請多組 API Key 並貼入設定中（以換行分隔），系統會自動進行負載平衡，將出題效率最大化。

Q: 素養題生成速度好像比較慢？

A: 是的。素養題需要 AI 先行構思真實生活情境，並進行跨段落的邏輯推理，運算時間會比傳統知識型題目稍長，請耐心等候精采內容。

Q: 我生成的題目可以分享給其他老師嗎？

A: 當然！點擊「上傳至題庫大廳」，您的智慧結晶就能嘉惠社群；同時您也可以在大廳中搜尋並下載其他老師分享的優秀考卷。
