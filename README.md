# QuestWiz - AI 智能命題神器 (v9.0)

QuestWiz 是一款專為教師與教育工作者打造的 AI 輔助命題工具。透過 Google Gemini API 的強大語意理解能力，協助教師將枯燥的教學素材快速轉化為具備深度、層次分明的專業測驗題庫。

---

## 🌟 核心功能

*   **多模態內容匯入**：
    *   **文字與檔案**：直接貼上文章或上傳 PDF 文件。
    *   **影像辨識**：上傳課本照片或講義截圖，AI 會自動辨識文字內容。
    *   **網址擷取**：貼上新聞、部落格網址，甚至 YouTube 影片連結（自動抓取字幕）。
*   **布魯姆 (Bloom) 認知層次控管**：
    *   可精確指定「記憶、理解、應用、分析、評鑑、創造」各層次的題數。
    *   支援自動配置演算法，確保題目難度與深度的均衡。
*   **雙重命題風格**：
    *   **知識記憶型**：著重於事實陳述與概念理解。
    *   **素養導向型**：結合情境設計，評量學生解決問題的綜合能力。
*   **全方位編輯器**：
    *   生成後即時預覽，支援拖曳排序、複製、刪除。
    *   可手動微調題目敘述、選項與正確答案。
*   **多元匯出格式**：
    *   **考卷列印**：產生美觀的 A4 PDF 考卷。
    *   **數位學習**：支援 Blooket、Gimkit 互動平台專用 CSV。
    *   **備份分享**：純文字檔匯出或上傳至社區題庫大廳。

## 🚀 使用說明

### 1. 初始設定
*   開啟網頁後，點擊右上角的**「設定」**（齒輪圖示）。
*   在 API 金鑰欄位填入您的 **Google Gemini API Key**。
*   設定完成後，金鑰會安全地儲存在您的瀏覽器中（有效期限為 2 小時）。

### 2. 提供素材
*   選擇輸入分頁：**「文字/檔案」**、**「影像」**或**「網址」**。
*   提供您想要 AI 出題的來源內容。

### 3. 配置生成參數
*   **選擇學生程度**：例如國中一年級、高中三年級等。
*   **設定出題風格**：選擇「知識記憶型」或「素養導向型」。
*   **設定層次分配**：展開「布魯姆認知層次」選單，勾選您想要的層次（若不勾選，系統會自動分配）。
*   **設定數量與難度**：決定要產生的題目總數。

### 4. 開始生成與編輯
*   按下**「開始出題」**按鈕，等待 AI 處理（素養題因為情境複雜，生成時間會較長）。
*   題目出現後，您可以在右側預覽區直接修改內容。
*   使用左側的 **「≡」符號** 進行拖曳排序。

### 5. 下載與分享
*   滿意後點擊**「下載題庫檔案」**，選擇您需要的格式。
*   您也可以點擊**「上傳至題庫大廳」**，將這份優質考卷分享給其他老師。

## 🛠️ 技術支援與部署

### 本機測試
直接在 `stable` 資料夾中開啟 `index.html` 即可運行。

### 部署至 Netlify
1. 登入 Netlify 帳號。
2. 將整個 `stable` 資料夾拖入部署區域。
3. 您的線上命題系統即刻上線！
